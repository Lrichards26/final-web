Welcome to Our Final Project

For our final project, we wanted to create a single-player game that allowed people to choose their character. We chose to use our Bitmojis as characters, who go through different obstacles and end up meeting up with each other. To play this game, a player uses the arrows on a keyboard to try their best to get through as many obstacles as they can and make it to the end! This JavaScript code utilizes several arrays and functions to create a two-player game environment. The game involves navigating Bitmoji characters through obstacle courses, with different phases and win/lose conditions. The drawing functions (`DrawShirt`, `DrawHead`, `DrawBitmoji`, `drawBitmoji`) generate visual representations of the characters. Three obstacle-drawing functions (`Obstacle1`, `Obstacle2`, `Obstacle3`) contribute to randomly generated obstacle courses. Arrays (`round1` and `round2`) store course information, and the `generateCourseNumbers` function populates these arrays with randomly selected obstacle types. The `CourseCreate` function then renders the obstacle courses on the screen. The game's main loop (`draw`) manages player input, obstacle collisions, and game phases, transitioning between stages based on gate activation and collisions. Additionally, user interface elements include buttons (`startButton` and `restartButton`) for initiating or restarting the game, with mouse clicks triggering corresponding actions. Overall, the code demonstrates the integration of arrays, functions, and p5.js graphics to create an interactive and dynamic gaming experience.
